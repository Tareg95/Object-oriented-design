# 🧱 Object-Oriented Design (KTH)

## 📌 Overview
This project was developed as part of a course at KTH focusing on object-oriented programming in Java.

The goal was to practice:
- Class design
- Encapsulation
- Inheritance
- Modular code structure

---

## 🧠 Key Concepts
- Object-oriented design principles  
- Separation of concerns  
- Reusable components  

---

## 🛠️ Language
- Java

---

## 👨‍💻 Note
This project reflects coursework and focuses on fundamental OOP concepts rather than production-level design.
